火狐浏览器使用笔记

2026-06-12
字数统计: 575   |   阅读时长≈ 2 分钟

我转到了火狐浏览器，谷歌备用。

安装和迁移数据

这里说一下细节，关于重复性的问题

  • 书签 : 不考虑重复的问题，迁移过来的书签直接追加，因此可能产生重复书签。因此不建议来回倒书签。

  • 密码：考虑重复，但是密码导入不是"同者更新"，而是"同者跳过"。 它宁可留着你旧密码不动，也不冒险用 CSV 里可能过期的密码覆盖你浏览器里正在用的。具体规则如下

    情况 Firefox 怎么做 Chrome 怎么做
    URL+用户名+密码都一样 ✅ 识别为重复 → 跳过（不产生两份） 通常跳过
    URL+用户名同，但密码不同 ⚠️ 也当成重复 → 跳过不改（这就是 bug 说的"不会更新"） 行为版本相关，可能跳过也可能多一条
    URL 全新的 ✅ 追加进来 ✅ 追加进来

因此，最好选择好一个浏览器作为主浏览器，所有数据都放在一个浏览器中。另一个只负责接收"最终版 HTML/CSV"一次，平时别来回倒。

隐私保护

让 Firefox 的“防追踪”处于你想要的水平（而不是盲目最强）

去：

Settings → Privacy & Security → Enhanced Tracking Protection

  • 选 Strict（严格）（更偏你要的“少被跟踪”）

现实预期：

  • Strict 档：很多新闻站、视频站、登录态会多让你点一次“允许/刷新”
  • Standard 档：更省心，但更“温和”

你如果是日常主力，我建议先从 Strict 跑一周；真有站坏再按域放行（点锁图标 → 关闭保护）。

装一个真正的护城河：uBlock Origin（不是 AdBlock）

Firefox 插件商店搜 uBlock Origin（作者：Raymond Hill，蓝盾图标）：

  • 装完默认已经很强
  • 想要更狠：Dashboard → Filter lists → 勾 Privacy / Malware domains / Annoyances（Annoyances 会把一些弹窗/横幅逻辑砍得更狠，偶尔也砍过头）（没看到在哪）

别同时装多个同类拦截器（例如 AdBlock + uBlock），会互相打架。

总结

  1. 隐私保护：Strict（我还是改成了标准，不然网页老不让跳转，总得确认点击一下）
  2. uBlock Origin：装，默认开
  3. 别在 Firefox 里登 Google 账号同步
  4. Chrome 备选：用完关掉，别让它慢慢又变成你主浏览器
