A股股票交易手续费

https://zhuanlan.zhihu.com/p/38360510

印花税

股票交易金额的千分之一（0.1%），只在股票卖出的时候收取。

过户费

过户费为成交股票数量的万分之六（0.06%），不足1元按1元计算（要买到16手以上，过户费才高于1元），买入和卖出均要收取。

目前上交所（60开头）收取，深交所不收取。

佣金

按交易金额的一定比例收取，一般不足5元按5元收费。

买入和卖出均收取。

小结

过户费基本很少，可以忽略不记，目前我就是买入卖出就2块（深交所还不用出）（额，我看了我的交割单，过户费是0.5元）。佣金免五后收取金额也很少（万分之一，买入卖出平均一万也就2块）。

成本最高的是印花税，以一万为例，卖出就10块。

也就是说目前我买入股票手续费很低，卖出要交千分之一的印花税，其它可忽略不计。

话说回来，都2023年了，我股票到现在都还套着呢，当然我买股票也是出于玩票的心理。我深刻地认识到，一般人估计都很难从A股挣到什么钱，总体算下来不赔钱就算不错了。

分红概念

https://www.zhihu.com/question/314361762/answer/1061573703

在股权登记日（即统计和确认参加股息红利的日期）收盘前购买了股票就可以享有本期股票分红。

一般分红后第二天会进行除息，因此投资者的总资产没有增加，而且因为分红需要缴税，总资产甚至是下降的。

分红的意义就在于大肥羊得到了实实在在的现金，而投资者持有的股票数量没有发生变化。

红利税

红利税不是拿到分红时缴纳的，而是在拿到分红后，卖出「获得分红的股数」的第二天，由证券公司代扣代缴。

  1. 没有拿到分红，不用担心有红利税支出，只有拿到分红后，才需要考虑红利税；

  2. 分红时不会立即支出红利税，而是在卖出这些拿了分红的股数后，第二天由证券公司进行代扣代缴。

股票分红扣税跟股票的持有时间长短有关，且分红后不会立马扣税，待卖出时才扣税。

1、持股期限在1个月（含）以内的，卖出时扣除分红所得的20%作为个人所得税；

2、持股期限在1个月以上至1年（含）的，卖出时扣除分红所得的10%作为个人所得税；

3、持股期限超过1年的，不需要缴纳个人所得税；

4、股票分多次买入的，一律按照先进先出原则一一对应计算持股时间；

5、当日有买进卖出的（即所谓做T），收盘后系统计算你当日净额，净额为买入，则记录为今日新买入，净额为卖出，则按照先进先出原则，算成你卖出了你最早买入的对应数量持股，并考虑是否扣税和税率问题。

6、以上扣除的税会在股票卖出时由券商直接扣除。

简单总结一下：没有拿到分红的股票不用扣税，拿到分红的股票不是直接扣税，而是卖出时扣税（两个链接有一点出入在于是卖出时直接扣除红利税还是第二天扣除）；做T看当日净额；先进先出原则；持股超一年，不用缴税。

ETF 与 LOF 基金的区别

https://zhuanlan.zhihu.com/p/37425870

ETF即交易型开放式指数基金，通常又被称为交易所交易基金（Exchange Traded Funds），是一种在交易所上市交易的、基金份额可变的一种开放式基金。

LOF，英文全称是"ListedOpen-EndedFund",汉语称为"上市型开放式基金"。也就是上市型开放式基金发行结束后，投资者既可以在指定网点申购与赎回基金份额,也可以在交易所买卖该基金。

ETF

ETF基金有两种交易方式，第一种是像股票一样买卖，买卖仅收佣金费用；第二种是用一揽子股票申购赎回，在做ETF申购时，投资者需要提前购买后指数对应的一篮子股票，然后用一篮子股票来申购ETF的份额；赎回时，投资者拿到手的也不是现金，而是对应的一堆股票，申购赎回费用没找到资料。第二种方式对现金量有要求，一般要50万以上，所以目前与我无关。一般人只能是第一种方式，类似于买卖股票。

其实ETF基金还有一个费用，那就是运行费用。比如 300ETF (SH510300)的费用是 管理费：0.5%，托管费：0.1%，指数使用费：0.03%。**这些费用会在每天的净值中直接减去。**可以查你想买的ETF的招募说明书查看具体的费率。也就是说，ETF基金默认扣除了年化0.6%的收益，这个比例还可以。

ETF基金的特点：

  1. 只能场内交易

  2. 跟踪指数

  3. 几乎百分百买股票，现金很少，比如512690股票比例是97.65%

  4. 运营费用低

LOF

LOF基金也一样是两种交易方式，第一种是场内买卖，和ETF一样，买卖只收佣金；第二种是场外申购赎回，直接现金交易，也就是说投资者是用现金买基金份额，然后用份额赎回得到现金，另外门槛低，1000起步。

场内和场外购买，管理费用一样。管理费和托管费都是在基金资产里计提的，二级市场里买卖的都是扣费后的。

https://www.zhihu.com/question/56018270

LOF 特点：

  1. 交易场所既可以是场内，也可以是场外
  2. 无法满仓。由于需要应对赎回，一般股票比例95%
  3. 运营费用相对ETF较高。LOF的管理费大部分为0.75%，托管费为0.15%。（这个数据旧了，以富国天惠为例，管理费1.5%，托管费0.25%。）这是因为LOF基金申赎时需要买卖股票，从而提高了交易成本。而ETF直接使用股票申赎的，交易成本低。
  4. 可以不跟踪指数，主动买股票

总的来说个人感觉LOF不如ETF。知乎上说LOF的优势就在于可以不开户，方便普通投资者购买，这个对我不成立。

小结

  • ETF和LOF 都是场内购买划算，主要是省了买入卖出的费用（运营费用不变）。
  • **用途：**ETF只能追踪指数，而LOF可以是主动型基金。
  • 费用： ETF和LOF 买入卖出都只收佣金，ETF运营费用很低，而且好像基本都是0.6%; LOF不同基金收费不同（具体查看该基金的招募说明书），但比ETF高
  • 风险：ETF 风险更低，因为是宽基基金，哪怕某个股票暴雷也不影响；LOF就不同，这个就要看基金经理了。

税务管理

个人所得税

https://www.youtube.com/watch?v=NanTBuduuiA

本文为看李永乐老师视频的笔记

应纳税所得额

应纳税所得额 = 综合所得额 - 免征 - 扣除

  • 综合所得额 ：工资 + 劳务费×80% + 稿费×80%×70% + 特许使用费×80%

    劳务费大概就是做副业的钱，特许使用费大概是把版权借给其他公司使用的钱

  • 免征：目前是一年6万，所有人都一样

  • 扣除：分为专项扣除和附加扣除

    1. 专项：三险一金等

    2. 附加：子女教育（1.2万/年）

         	继续教育（考证，4800/年，考上了3600/年）

   		大病医疗（自付>1.5万，超出部分不交税，最多8万）

         房贷（必须首套，定额，1.2万/年）

   		 房租（省会、直辖市等1.8万，普通城市1.32万）（房租和房贷只能选一个）

         赡养老人（独生子女2.4万/年，非独生最多1.2万/年)

所得税计算

按应纳税所得额划分不同的税率。累进税制。

比如你应纳税所得额为5万，那么你前3.6万按照3%计算，后1.4万按照10%计算。

汇算清缴

工资

工资：收入 - 免征 - 扣除 （公司预扣的所得税）

补加个人所得税的情况：假如一个人一年内同时在两家公司工作，两家公司只按他在本公司的工资缴税，就导致他的累进税率较低。

劳务

退税主要是劳务费，劳务费的应纳税收入计算方法为，如果劳务报酬不超过800为0，800-4000时则为劳务费-800元，超过4000为劳务费×80%。

劳务费预扣税率为：应纳税所得额不超过20000为20%，超过2万至5万的部分为30%，超过5万的部分为40%。

劳务税的预扣税率比工资高得多。

清缴

第一种情况如果你换了工作，一年内在两家公司工作，那么你可能要补加税费。但是可以通过填附加扣除降低税费。

劳务费就可以退税。

不需申报的情况

  1. 年收入 ≤ 12万
  2. 补税 ≤ 400

年终奖计税

年终奖有两种计税方式，第一种是单列，第二种计入汇总中，两种可能有差异，视情况来选择。

保险

我个人只是买了一个百万医疗险，买保险需要认真查看相关的保险内容，看看自己符不符合全部条件，尽量避免需要出险的时候不能赔偿。

理赔时保险公司调查健康状况

https://zhuanlan.zhihu.com/p/90282213

保险公司考虑到成本，一般不会在投保前进行严格的调查，如果年龄、地区、职业符合要求，并且健康告知通过，那么一般都可以正常投保。除非发生了高保额、财务状况与保额不符的情况，保险公司才会注意到投保人。

但是投保前不查并不代表理赔时不查，毕竟一个是你给保险公司交钱，另一个就是保险公司给你钱。这也就是我们所说的买保险时的“宽进严出”——买保险的时候容易买，可是理赔的时候就没有那么容易理赔了。

所以，理赔时的调查，才是重中之重。

保险公司一般会根据理赔金额的大小、投保时间与申请理赔时间的长短、是否存在骗保的可能性，来确定是否进行更详尽的审查。

一般来讲，保险公司有3种方式获知投保人的既往病史：

a）调查社保就医记录

b）调查医院门诊住院记录

c）调查同业的投保和理赔记录
