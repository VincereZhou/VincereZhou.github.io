软件CrystalDiskInfo_查看硬盘健康

2026-04-16
字数统计: 891   |   阅读时长≈ 3 分钟

下面内容来自DEEPSEEK，我用这个软件查看硬盘的健康程度。

安装与使用

从 官网 下载 ZIP 压缩绿色版本，解压后直接运行 DiskInfo64.exe文件，不用安装，软件界面就是你电脑现在所有硬盘的健康程度信息。

如何看结果

用 CrystalDiskInfo 看硬盘，最直观的判断逻辑是：先看“脸”（顶部状态栏），再看“内脏”（SMART 具体项）。机械硬盘（HDD）怕“坏道”，固态硬盘（SSD）怕“爆 0E”和“写死”。

一、通用第一眼：看“脸”

无论机械还是固态，打开软件先看顶部这几行，这是最快的健康筛查：

位置 看什么 正常状态 异常动作
健康状态 颜色/文字 蓝色 + “良好” 黄色/红色 = 立刻备份数据
温度 数值 (°C) HDD < 45℃ SSD < 60℃ 长期 > 70℃ 会严重折寿
通电时间 09 属性 HDD > 3万小时算高龄 SSD 看写入量更准 结合具体指标判断

二、机械硬盘（HDD）：盯着“坏道”看

机械硬盘是物理磁头读写，最怕盘片划伤。请直接在软件下方的 SMART 列表中找到这三个 ID（注意看 ID 号，因为不同品牌名称可能不同）：

关键 ID 属性名（常见） 核心含义 判断标准（原始值）
05 重映射扇区计数 已经发现的坏道，被替换到备用区 必须 = 0。数值越大，盘片物理损伤越严重。
C5 当前待映射扇区 疑似坏道，正在排队等修复 必须 = 0。只要 > 0，就是故障前兆，需警惕。
C6 无法修正的扇区 彻底读不出来的坏道 必须 = 0。一旦 > 0，说明数据已经丢失。

HDD 速查口诀：05、C5、C6 这三个原始值必须全为 0。任何一个非零，尤其是 C5/C6 增长，请立即备份。

三、固态硬盘（SSD）：盯着“0E”和“寿命”看

固态硬盘没有机械部件，但怕闪存颗粒损坏和写入磨损。重点关注以下指标：

这里后面两个不用看，顶部已经告诉了健康度和主机写入量，而且对于按照我的使用习惯，主机写入量也不用管，基本上不会达到厂家的设定上限。

关键 ID 属性名（常见） 核心含义 判断标准（原始值）
0E 媒体与数据完整性错误 闪存颗粒出现不可修复的错误 必须 = 0。这是 SSD 的“死刑”指标，只要 > 0，立刻停用换盘。
03 可用备用空间 剩余备用块百分比 新盘 = 100。持续下降说明坏块增多，低于 10 很危险。
05 / B1 已用寿命百分比 磨损度（100-当前值=健康度） 新盘 = 0。越接近 100 越危险，建议 > 80 时准备换盘。
F1 / E9 主机写入量 累计写入数据量 对比厂商 TBW 规格，判断消耗进度。

SSD 速查口诀：0E 必须为 0。如果 0E > 0 或 健康度（100 - 05）< 20%，盘就不安全了。

四、总结

硬盘类型 第一眼（顶部） 必看 ID（列表） “死刑”信号
机械 HDD 状态蓝色、温度正常 05, C5, C6 05/C5/C6 任一数值 > 0 且增长
固态 SSD 状态蓝色、温度正常 健康度, 0E, 03 0E > 0 或 健康度 < 10%

操作建议：建议每 1-2 个月打开软件截个图存档，对比关键 ID 的数值是否有增长趋势。趋势恶化比绝对值更可怕。
